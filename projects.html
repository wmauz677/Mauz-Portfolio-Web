Projects

MyCaddie

During the summer of 2017 I worked with my good friend Kevin Liao on our first IOS app. Neither of us had any experience with app building, but we decided to give it our best shot. We have finished the app and it will be uploaded to the app store by January 1st, 2018.

The app uses a Firebase database to keep user data on the cloud. It was programmed in Xcode using the language Swift. Design work for the app was done using Sketch. The repository of the app can be found on Github.

Check out the app's website: MyCaddie Website

Here is the repository on Github: MyCaddie Repository

Gaming Glove

During the Spring semester of 2016, I made a video-gaming glove with my partner for the class, John Hickey. Our task was to create something using Matlab, Arduino, and four sensors.

The Arduino Uno was loaded with our code from Matlab that utilized a customized library in order to communicate with the serial monitor. From here we were able to create games on-board the glove, along with games it could interact with on the computer.
